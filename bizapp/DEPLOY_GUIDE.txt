# BizHisaab — Online Deploy Karne ka Tareeqa
# =============================================
# Railway.app pe FREE mein deploy karein
# (Credit card nahi chahiye pehle $5 tak free)

## STEP 1 — GitHub Account Banayein
1. https://github.com par jayen
2. Sign Up karein (bilkul free)

## STEP 2 — Ye Files GitHub pe Upload Karein
1. GitHub pe "New Repository" click karein
2. Naam: bizhisaab
3. Tamam files upload karein:
   - app.py
   - requirements.txt
   - Procfile
   - templates/ folder (sari files)

## STEP 3 — Railway pe Deploy Karein
1. https://railway.app par jayen
2. "Login with GitHub" se login karein
3. "New Project" → "Deploy from GitHub repo"
4. apna bizhisaab repo chunein
5. Railway khud deploy kar dega — 2-3 minute mein

## STEP 4 — URL Milega
- Railway ek URL dega jaise: bizhisaab-production.up.railway.app
- Ye URL aap aur employees dono use kar sakte hain
- Mobile pe bhi chalega

## EMPLOYEES KO ADD KARNA
1. Admin login karein

3. Employee ka naam, username, password set karein
4. URL share karein employees ke saath

## COST
- Railway Free Plan: $5/month free credit (kaafi hai small business ke liye)
- Paid Plan: $5/month (~Rs 1400) — unlimited

## LOCAL TEST (pehle apne computer pe test karein)
1. Python install karein
2. Command prompt mein:
   cd bizhisaab-folder
   pip install flask werkzeug
   python app.py
3. Browser mein: http://localhost:5000
